# Trader Performance Analysis using Bitcoin Fear & Greed Index

## About the Project

This project was completed as part of the Primetrade.ai Data Science internship assignment.

The objective was to analyze historical Hyperliquid trading data along with the Bitcoin Fear & Greed Index to understand whether market sentiment has any impact on trader performance.

---

## Datasets Used

- Historical Hyperliquid Trading Data
- Bitcoin Fear & Greed Index

---

## Steps Performed

- Loaded both datasets using Pandas.
- Explored the data and checked for missing values and duplicates.
- Converted Unix timestamps into datetime format.
- Merged both datasets using the Date column.
- Created additional features for analysis.
- Performed exploratory data analysis (EDA).
- Visualized trader performance under different market sentiments.
- Summarized key findings and recommendations.

---

## Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Key Findings

- Traders achieved the highest average Closed PnL during Greed market conditions.
- The highest trading activity was observed during Fear.
- Win rate was highest during Extreme Greed.
- Trading behaviour and performance varied across different market sentiments.
